プライバシーポリシー

OniTan におけるデータの取り扱い方針を説明します。 本ポリシーは、iPhone アプリ OniTan に適用されます。

OniTan 開発者

1. 取得する情報

OniTan は、学習進捗、正答率、連続学習日数、寄付購入状態など、 アプリの機能提供に必要な情報を端末内に保存します。

アプリ利用のために、氏名、メールアドレス、アカウント登録を必須とはしていません。

2. 保存場所

学習進捗や設定情報は、主に iPhone 端末内の保存領域に保持されます。 開発者がそれらの内容を個別に受け取る仕組みはありません。

3. 課金について

寄付購入や購入復元は Apple の App Store の仕組みを通じて処理されます。 決済情報の管理は Apple のポリシーに従います。

4. 広告について

アプリ内で広告を表示する場合、広告配信のために第三者サービスを利用することがあります。 その場合、広告識別子、端末情報、利用状況、診断情報などが広告配信事業者によって収集されることがあります。

具体的な収集内容は、実際に利用する広告 SDK とその設定に依存します。 App Store のプライバシー表示は、その時点の実装に合わせて更新します。

5. 外部送信と第三者提供

アプリ機能上必要な範囲を除き、開発者が利用者の個人情報を第三者へ販売することはありません。 ただし、App Store 決済や広告配信など、外部サービスの利用に伴う情報処理は各事業者の方針に従います。

6. お問い合わせ

本ポリシーに関する問い合わせは GitHub Issues から受け付けます。

7. 改定

本ポリシーは、機能追加、法令対応、外部サービス変更に応じて改定することがあります。 重要な変更がある場合は、アプリまたは公開ページ上で案内します。
