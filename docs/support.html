ONITAN SUPPORT

サポート

不具合報告、改善要望、広告や寄付機能に関する問い合わせ先をまとめています。

GitHub Issues を開く プライバシーポリシー
問い合わせ先

現在のサポート窓口は GitHub Issues です。利用端末、iOS バージョン、再現手順があると確認が早くなります。

よくある内容

  • 問題データの誤字、誤答、説明文の修正依頼
  • 学習進捗が期待どおり保存されない場合の報告
  • 寄付購入や購入復元が反映されない場合の相談
  • 広告表示の不具合や、広告非表示が反映されない場合の報告

問い合わせ時にあると助かる情報

  • 利用している iPhone の機種名
  • iOS のバージョン
  • OniTan のバージョンとビルド番号
  • 不具合が起きるまでの手順
  • 表示されているメッセージやスクリーンショット

補足

App Store での課金処理は Apple の仕組みを通じて行われます。 返金や請求に関する最終的な手続きは Apple の案内に従ってください。
